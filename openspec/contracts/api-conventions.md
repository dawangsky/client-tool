# API 通用约定

## 基础

| 项 | 约定 |
|----|------|
| 风格 | RESTful JSON |
| 前缀 | `/api/v1` |
| 编码 | UTF-8 |
| 时间 | ISO-8601，`Asia/Shanghai` 业务日 |
| 金额 | 字符串或整数分（**团队选定一种，全项目统一**）；本文档示例用 **分（cent）** |
| 份额 | `decimal(18,4)` 序列化为字符串 |

## 认证

| 场景 | 方式 |
|------|------|
| C 端 | `Authorization: Bearer <access_token>` |
| 运营后台 | 同上 + 角色 `ADMIN` / `OPS` |
| 内部批处理 | 服务间密钥或内网白名单（实现期定） |

## 统一响应

```json
{
  "code": "0",
  "message": "ok",
  "data": {},
  "traceId": "uuid"
}
```

## 错误码（节选）

| code | HTTP | 含义 |
|------|------|------|
| `0` | 200 | 成功 |
| `40001` | 400 | 参数校验失败 |
| `40101` | 401 | 未登录 |
| `40301` | 403 | 无权限 |
| `40401` | 404 | 资源不存在 |
| `40901` | 409 | 幂等冲突（重复提交） |
| `42201` | 422 | 适当性不匹配 |
| `42202` | 422 | 产品不可交易（状态/限额） |
| `42203` | 422 | 份额不足 |
| `42204` | 422 | 已过截点不可撤单 |
| `50001` | 500 | 系统错误 |
| `50201` | 502 | 支付通道异常 |
| `50202` | 502 | TA 网关异常 |

## 幂等

- 写操作 Header：`Idempotency-Key: <uuid>`（客户端生成）
- 服务端表：`idempotent_key`，24h 内同 key 返回同一结果

## 分页

请求：`page=1&pageSize=20`  
响应：

```json
{
  "list": [],
  "total": 100,
  "page": 1,
  "pageSize": 20
}
```

## 贯穿键

所有交易相关接口必须能关联：

- `order_no`
- `trade_account_id`
- 涉及 TA 时：`ta_code`、`fund_account_no`

详见各模块 spec。
